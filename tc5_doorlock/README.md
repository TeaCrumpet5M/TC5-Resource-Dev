# tc5_doorlock v4

Added roller shutter support for garage doors.

## New
- door type selector in admin UI:
  - standard
  - roller
- roller open ratio setting
- roller doors use door system open ratio when unlocked

## Commands
- /tc5_doorui
- /tc5_createdoor
- /tc5_dooredit [doorId]
- /tc5_toggledoor

## Keybinds
- E = quick toggle nearest door
- LEFT ALT = tc5_eye interaction on configured doors
